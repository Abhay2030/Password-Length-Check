# Simple Password Checker 🛡️

A very lightweight, easy-to-understand password strength checker built with plain HTML, CSS, and JavaScript in a single file. 

## Features
* **Real-time checking:** Evaluates the password as you type.
* **Simple Logic:** Checks for length (8+ characters), numbers, and capital letters.
* **No external libraries:** Completely standalone and fast.

## How to use
1. Download or clone this repository.
2. Double-click the `index.html` file to open it in any web browser.
3. Start typing to see your password strength!
